Formulas and Resources

Creatinine Clearance (ml/min)

Cockraft-Gault Equation:
(140-Age) x weight(kg) x factor ÷ serum creatinine
factor if male = 1.23 factor if female = 1.04

Ideal Body Weight (kg)

  • Men: 50 + (2.3 x every inch over 5ft)
  • Women: 45.5 + (2.3 x every inch over 5ft)

Adjusted body weight

ABW(kg) = IBW + 0.4(Total Body Weight - IBW)

Body Surface Area

BSA (m2) = (height (cm) x weight (kg) ÷ 3600

Monofer Dose Calculation

Iron need(mg iron) = Body weight(kg) x (target Hb(g/dl) - Actual Hb(d/gl) ) x 2.4 + iron for iron stores (mg iron)
